Theme Documentation

PixelPerfect — Personal Portfolio Theme v1.0.0

Last updated: June 16, 2026

Introduction

PixelPerfect is a modern, premium personal portfolio theme designed for creative professionals, designers, and developers. Built with cutting-edge web technologies including Three.js for immersive 3D experiences, Bootstrap 5 for responsive layouts, GSAP for smooth animations, and AOS for scroll-triggered effects.

Theme Features

  • Interactive 3D Hero Background with Three.js
  • 5 Pre-built Pages: Home, About, Portfolio, Blog, Contact
  • Fully Responsive Design (Mobile, Tablet, Desktop)
  • Dark Theme with Gradient Accents
  • Custom Cursor Effects
  • Swiper.js Carousels for Services & Featured Projects
  • Enhanced Service Cards with badges, feature lists, pricing & tech tags
  • Enhanced Portfolio Cards with tech stacks & project metrics
  • 6 Project-Specific Detail Modals
  • Portfolio Filter System (on portfolio.html page)
  • Animated Skill Bars & Counters
  • "How I Work" Process Section
  • Client Logos Showcase
  • Contact Forms with Validation
  • GSAP Scroll Animations
  • SEO-Optimized HTML Structure
  • Cross-Browser Compatible
  • Comprehensive Documentation

Installation

Quick Setup

Follow these simple steps to get started with PixelPerfect:

  1. Unzip the downloaded theme package.
  2. Upload the entire pixelperfect folder to your web server's root directory (e.g., public_html, www, or htdocs).
  3. Open index.html in your browser to preview the theme.
  4. Edit the HTML files to customize content, images, and text.

Important Note

All libraries are loaded via CDN. Ensure you have an active internet connection for the theme to function properly. For offline use, download the libraries and update the paths in the HTML files.

Local Development

For local development, you can use any HTTP server. Here are a few options:

# Using Python
python -m http.server 8000

# Using Node.js (with npx)
npx serve pixelperfect

# Using VS Code
# Install the "Live Server" extension and click "Go Live"

CSS Customization

Main Stylesheet

The main stylesheet css/style.css is organized into clearly labeled sections:

  • CSS Variables — Colors, fonts, transitions, and shadows
  • Reset & Base — Basic element resets and global styles
  • Typography — Heading and text styles
  • Navigation — Navbar, mobile menu
  • Hero Section — 3D hero layout and stats
  • Components — Service cards, portfolio cards, blog cards
  • Sections — Testimonials, contact, footer
  • Utilities — Buttons, badges, animations

Modifying Colors

You can easily change the theme colors by editing the CSS variables in :root:

/* File: css/style.css - Lines 5-30 */

:root {
    /* Primary Colors */
    --color-primary: #7c6ff0;        /* Main indigo */
    --color-primary-dark: #5f51d1;   /* Darker indigo */
    --color-primary-light: #b0a6f5;  /* Lighter indigo */
    
    /* Accent Colors */
    --color-secondary: #f0b27a;      /* Warm gold */
    --color-accent: #6dd5c8;         /* Mint */
    --color-accent-alt: #f0c27f;     /* Gold */
    
    /* Background Colors */
    --color-bg: #0b0a0d;             /* Main background */
    --color-bg-dark: #08070a;        /* Darker background */
    --color-bg-light: #121116;      /* Lighter background */
    --color-bg-card: #19181e;        /* Card background */
    
    /* Text Colors */
    --color-text: #eeedf0;           /* Main text */
    --color-text-muted: #8a8794;     /* Muted text */
}

Pro Tip

Search for --color- in style.css to find all color variables. Change a single variable to update the color across the entire theme.

Contact Form

Form Setup

The contact form is set up with client-side validation and a simulated send animation. To make it functional, replace the simulated AJAX call in js/main.js:

// In main.js - Find this section and replace with your backend endpoint
$('#contactForm, #contactFormMain').on('submit', function(e) {
    e.preventDefault();
    
    // Replace this with your AJAX call:
    $.ajax({
        url: 'https://your-api.com/send-message',
        method: 'POST',
        data: $(this).serialize(),
        success: function(response) {
            // Handle success
        },
        error: function(error) {
            // Handle error
        }
    });
});

Adding Form Fields

To add or remove form fields, edit the HTML form and update the validation logic in main.js.

Swiper Carousel Integration

How It Works

The theme uses Swiper.js 11 for touch-enabled, responsive carousels. Two carousels are initialized on the homepage:

  1. Services Carousel (.services-swiper) — 6 service cards, autoplay every 5 seconds
  2. Portfolio Carousel (.portfolio-swiper) — 6 project cards, autoplay every 4 seconds

Configuration

Both carousels are initialized in js/main.js under the "Swiper Initialization" section:

// Services Swiper
new Swiper('.services-swiper', {
    slidesPerView: 1,
    spaceBetween: 24,
    loop: false,
    autoplay: {
        delay: 5000,
        disableOnInteraction: true,
        pauseOnMouseEnter: true,
    },
    pagination: { el: '.services-pagination', clickable: true },
    navigation: {
        nextEl: '.services-next',
        prevEl: '.services-prev',
    },
    breakpoints: {
        576: { slidesPerView: 2 },
        992: { slidesPerView: 3 },
    },
});

// Portfolio Swiper
new Swiper('.portfolio-swiper', {
    slidesPerView: 1,
    spaceBetween: 24,
    loop: false,
    autoplay: {
        delay: 4000,
        disableOnInteraction: true,
        pauseOnMouseEnter: true,
    },
    pagination: { el: '.portfolio-pagination', clickable: true },
    navigation: {
        nextEl: '.portfolio-next',
        prevEl: '.portfolio-prev',
    },
    breakpoints: {
        576: { slidesPerView: 2 },
        992: { slidesPerView: 3 },
    },
});

Key Options

OptionDescriptionDefault
slidesPerViewNumber of slides visible at once1 (mobile), 2 (tablet), 3 (desktop)
spaceBetweenGap between slides in px24
autoplay.delayAuto-advance interval in ms4000–5000
pauseOnMouseEnterPause autoplay when hoveringtrue
clickableAllow clicking pagination dotstrue

Three.js Integration

How It Works

The 3D hero background is rendered in js/three-hero.js. It creates:

  • Particle System: 2,000 colored particles in a spherical distribution
  • Floating Geometries: 40 wireframe shapes (icosahedrons, octahedrons, torus, etc.)
  • Connecting Lines: Subtle point-based line system
  • Mouse Tracking: Particles and camera respond to mouse movement

Customizing the 3D Scene

Key configurable parameters in three-hero.js:

// Number of particles
const particleCount = 2000;

// Number of floating geometries
const geoCount = 40;

// Particle colors
const color1 = new THREE.Color('#7c6ff0'); // Primary
const color2 = new THREE.Color('#6dd5c8'); // Accent

// Particle size
particlesMaterial.size = 0.3;

// Camera distance
camera.position.z = 30;

Disabling the 3D Background

To remove the Three.js background, either:

  1. Remove the <div id="three-container"> from index.html
  2. Remove the <script src="js/three-hero.js"> script tag
  3. Or remove the Three.js CDN link from the head section

Performance Note

The 3D scene automatically disables on devices with prefers-reduced-motion: reduce enabled. Custom cursor effects also disable on mobile devices (below 992px).

Animations

Animation Libraries

The theme uses three animation libraries:

Library Used For How to Customize
AOS Scroll-triggered fade/slide animations data-aos="fade-up", data-aos-delay="100"
GSAP Advanced scroll-triggered animations, parallax In main.js GSAP section

AOS Usage

Add animations to any element using data attributes:

<div data-aos="fade-up" data-aos-delay="100" data-aos-duration="800">
    Content to animate
</div>

Available animations: fade-up, fade-down, fade-left, fade-right, zoom-in, zoom-out, flip-up, flip-down.

GSAP Animations

GSAP animations are configured in main.js. You can add custom animations:

// Example: Animate elements on scroll
gsap.from('.your-class', {
    scrollTrigger: {
        trigger: '.your-class',
        start: 'top 80%',
    },
    y: 60,
    opacity: 0,
    duration: 0.8,
    stagger: 0.1
});

Custom JavaScript

The js/main.js file is organized into clearly labeled sections. You can add your own JavaScript at the end of the file, inside the DOM ready handler:

(function($) {
    'use strict';
    
    $(document).ready(function() {
        
        // ==== EXISTING CODE ====
        
        // ==== YOUR CUSTOM CODE ====
        // Add your custom JavaScript here
        
    });
})(jQuery);

Key functions available in main.js:

  • animateCounters() — Animates stat numbers on scroll
  • animateSkillBars() — Animates skill bar widths on scroll
  • isElementInViewport(el) — Checks if an element is visible
